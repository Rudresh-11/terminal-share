How it Works

1. Run the Agent

Start the agent on your host machine. It will generate a unique 6-digit session code.

2. Share the Code

Give that 6-digit code to your collaborator or enter it here on the website.

3. Real-time Mirroring

The collaborator sees your terminal output and can send keystrokes back in real-time.
